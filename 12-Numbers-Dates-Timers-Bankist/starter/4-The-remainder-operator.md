## The remainder operator

- The remainder (%) operator returns the remainder left over when one operand is divided by a second operand. It always takes the sign of the dividend.
```
console.log(13 % 5);
// Expected output: 3

console.log(-13 % 5);
// Expected output: -3

console.log(4 % 2);
// Expected output: 0

console.log(-4 % 2);
// Expected output: -0
```
